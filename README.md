# Look.tg  

## Это исходный код с полной логикой сайта для поиска НФТ телеграм подарков
### Парсинг подарков
Вся логика парсинга подарков находится в папке /gdb. Подарки парсятся через kurigram, используются сессии юзерботов. Чем больше сессий, тем быстрее будут собираться новые подарки и обновляться старые. В момент работы сайта использовалось ~30 сессий, их хватало чтобы раз в сутки обновить все подарки (7-8кк) и собрать информацию о новых.<br>
Для их хранения используется бд postgresql.<br>
Остальная информация находится в самих файлах.
